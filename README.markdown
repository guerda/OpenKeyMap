OpenKeyMap
==

Introduction
=
A map that displays all OpenPGP keys listed on [biglumber.com](http://biglumber.com) on a map, based on the [OpenStreetMap](http://www.openstreetmap.org).

Requirements
=
Python 3.x
[BeautifulSoup](http://pypi.python.org/pypi/BeautifulSoup)
